HOTEL REVIEW SYSTEM

Modules

  1. Landing Page - List of all hotels
  2. Hotel Details - A detailed description of the hotel including prices and comments
  3. Login/Sign Up Page - Adding new user / signing in
  4. Add review page - Add the new review to the website
  5. Add comments/ratings - Add comments/rating to the hotel details
